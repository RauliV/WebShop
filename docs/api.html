POST /api/products

Adds a new Product. Notice: _id is auto-generated by the server, do NOT add _id to the request body.

Path Parameters

None

Request Body

Product Inventory item to add.

Example Request

{
  "price" : 1.15,
  "name" : "Red 2*4 building block",
  "image" : "http://example.com/aeiou",
  "description" : "Classic Danish-style red 2*4 plastic building block"
}

Responses

201
Product product created
400
Error error message(s)

Example Response

Content-Type: application/json

{
  "_id": "2398d576bcaa672382ff2acf",
  "price" : 1.15,
  "name" : "Red 2*4 building block",
  "image" : "http://example.com/aeiou",
  "description" : "Classic Danish-style red 2*4 plastic building block"
}

404

POST /api/orders

Adds a new order. Notice: _id for the actual order and each order item should be auto-generated by the server, do NOT add _id to this request body except for each product. Also customerId is NOT part of the request and it is automatically set to the id of the current user.

None

Request Body

Order

Example Request

{
  "items": [
    {
      "product": {
        "_id": "98bcbbaf57632f21adf12301",

API Methods - Guests

Operations that do not require logging in

POST /api/register

Register a new user. Notice: _id is auto-generated by the server, do NOT add _id to this request body. All registered users will have their role forced to customer by the server

Path Parameters

None

Request Body

User

Example Request

{
  "name" : "Emmet Brickowski",
  "email" : "emmet.brickowski@google.com",
  "password" : "ZiOl3@8u9jskjed#76saMf=43sdfas"
}

Responses

201
User created user
400
Error error message(s)

Example Response

Content-Type: application/json

{
  "_id" : "f398d576bcaf672382ff2ac6",
  "name" : "Emmet Brickowski",
  "email" : "emmet.brickowski@google.com",
  "password" : "$2a$10$N9qo8uLOickgx2ZMRZoMyeIjZAgcfl7p92ldGxad68LJZdL17lhWy",
  "role" : "customer"
}
